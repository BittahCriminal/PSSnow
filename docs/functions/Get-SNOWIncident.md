# Get-SNOWIncident

## SYNOPSIS
Retrieves a servicenow Incident

## SYNTAX

```
Get-SNOWIncident [[-made_sla] <Boolean>] [[-number] <String>] [[-resolved_by] <String>] [[-opened_by] <String>]
 [[-state] <String>] [[-impact] <String>] [[-active] <Boolean>] [[-business_impact] <String>]
 [[-priority] <String>] [[-caller_id] <String>] [[-short_description] <String>] [[-assignment_group] <String>]
 [[-notify] <String>] [[-closed_by] <String>] [[-parent_incident] <String>] [[-contact_type] <String>]
 [[-reopened_by] <String>] [[-incident_state] <String>] [[-urgency] <String>] [[-company] <String>]
 [[-assigned_to] <String>] [[-severity] <String>] [[-approval] <String>] [[-reopen_count] <String>]
 [[-escalation] <String>] [[-location] <String>] [[-category] <String>] [-Sys_ID <String>] [-Query <String>]
 [-Fields <Array>] [-DisplayValue <String>] [-ExcludeReferenceLinks] [-Offset <Int32>] [-Limit <Int32>]
 [-RestrictDomain] [-SysParmView <String>] [<CommonParameters>]
```

## DESCRIPTION
Gets a record from the incident table

## EXAMPLES

### EXAMPLE 1
```powershell
Get-SNOWIncident -limit 1 -verbose
```

Returns a single Incident
